# Book Library

This repository contains a simple Book Library app for managing and tracking books in one place.

## What The App Does

The app lets you add books, view your list, update read status, edit existing entries, and remove books you no longer need. It also uses browser `localStorage` so your saved books stay available after refreshing the page.

## Tech Stack

- HTML
- CSS
- JavaScript
- Browser `localStorage`

## Purpose

I built this project as a practical tool to organize my reading list and track progress. It was also a way to practice core front-end skills by building a complete app without external frameworks.
