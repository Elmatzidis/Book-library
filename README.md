# Book Library

Book Library is a small web app for keeping track of books in one place. You can add a book with its title, author, and page count, mark it as read or not read, edit existing entries, and remove books you no longer want in the list. The app is designed to make a personal reading list simple to manage.

The project is built with plain HTML, CSS, and JavaScript. HTML handles the structure of the page and form, CSS controls the layout and styling, and JavaScript powers the app logic such as creating book cards, updating read status, editing entries, and storing data in `localStorage` so books remain available after a refresh.

I built this as a practical tool to manage and track books while practicing core front-end development skills. The main goal is to provide a clean, straightforward way to organize reading progress without relying on external frameworks or back-end services.
